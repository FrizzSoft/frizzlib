FilterOutCombosWithTheseDigits(List<string>, string)

Takes a list of digit combinations and removes any that include specified digits.

public static List<string> FilterOutCombosWithTheseDigits(List<string> Combos, string nonAllowedDigits)

Parameters

Combos List<string>

The list of combinations to process.

nonAllowedDigits string

The digits that are not allowed to occur in a combination.

Returns

List<string>

A new list containing only the combinations that do not contain non-allowed digits.

Remarks

In Sum Sudoku some digits may not be allowed in a cage due to the fact they already appear in an intersecting column, row or 3x3 block.

GetCombos(int, int)

Determines all possible combinations of digits given the sum of the digits and the number of digits.

public static List<string> GetCombos(int totalReqd, int cellsAvail)

Parameters

totalReqd int

The sum the digits must add to.

cellsAvail int

The number of digits to add together.

Returns

List<string>

A list of strings, each containing a possible combination. The list will be empty (have zero members) if no combinations are possible with the given constraints.

Remarks

In Sudoku the only digits allowed are the digits 1-9, and none may be repeated in a row, column, 3x3 block, or cage (in Sum Sudoku).
